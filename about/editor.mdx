## 工作流

### 启动预览环境

* 启用本地预览：`cd site && npm run dev`，打开 `http://127.0.0.1:3000` 实时预览。
* 监听文件改动：`./tools/bin/watch`，当本地 mdx 源文件修改时，自动同步到预览服务中。

### 提交前检查

正在开发中。

执行 `./tools/bin/lint` 进行风格检查。

自动风格检查目前启用了以下工具：

* markdown-lint

### 提交

Fork 仓库到个人账号下，从最新的上游内容创建新分支，提交 Pull Request 给上游。
