# 内部开发者门户

- **缩写**：IDP
- **英文名称**：Internal Developer Portal

## 定义

内部开发者门户（Internal Developer Portal）是一个集中化的 Web 界面，为开发者提供访问内部开发者平台的入口。它是开发者与平台能力之间的主要交互界面。

关键特性：

- **服务目录**：展示组织内所有可用的服务、API 和工具
- **软件模板**：提供项目脚手架和最佳实践模板
- **文档中心**：集成技术文档、API 文档和使用指南
- **自助服务**：允许开发者自主创建、配置和管理资源
- **可视化仪表板**：显示服务健康状态、性能指标和成本信息
- **搜索功能**：快速查找服务、文档和资源

与内部开发者平台（IDP）的区别：
- **平台**：底层的技术基础设施和服务
- **门户**：与平台交互的用户界面层

常见的开发者门户工具包括 Backstage、Port、Cortex 等。
